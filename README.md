# McpShortcuts

McpShortcuts 是一个基于 Android 的 MCP 客户端。你可以用它连接 MCP 服务器，调用工具，接收服务器通知。以及在应用内的 AI 聊天中使用这些工具。

## 核心功能

- 连接 MCP 服务器。
- 调用 MCP 工具。
- 接受来自 MCP 服务器的通知。
- 通过 OpenAI 风格的 API ，与 LLM 聊天。
- 在 AI 聊天中调用工具。

## 使用方法

1. 添加你的 MCP 服务器。支持 HTTP Streamable 的 MCP 服务器。可以自定义 HTTP Header ，如需身份验证，可以在这里设置。
2. 选择工具执行。在执行前输入参数，查看执行结果。
3. 查看并报错过往的执行记录，以便下次可以快速执行。

## AI 聊天

1. 在设置中打开 AI 聊天功能，配置 API URL 和密钥（配置在请求头上）。
2. 返回首页，从AI聊天入口中进入。
3. 右上角按钮即为切换工具调用是否为手动审批、自动处理。
